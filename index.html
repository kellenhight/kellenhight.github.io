What I'm Working On

Building a TOTEM Split Keyboard has been my most recent personal project, and it was an absolute blast to build. This open-sourced keyboard design was quirky enough to get my interest but has been a breeze to use thanks to its non-traditional key layout. I've found it far more comfortable than the traditional layout, and it introduced me to layered keymaps, which have been super useful and efficient. If you're into this sort of thing, here's my keymap. Building a few of these with some friends was a great introduction to the keyboard hobby, and I'm considering building a couple more for fun sometime; something with a pointing device or a knob would be fun.

Come back later! Cooler things are cooking...

Featured Web Links

These are some cool projects and resources I've found on the web, many of them Open Source. Give them a look if any pique your interest!

  • The TOTEM Split Keyboard was mentioned in the previous section, but bears repeating because it's such a high-quality build with solid documentation and value. Very fun starter keyboard and an overall solid piece of open hardware.
  • Manifesto for a Humane Web gave me words for the beliefs I have about how the internet (and all software/electronics) should be built: as tools for humans. In the spirit of this paper, here's a couple Humane Web-related platforms that provide alternatives to the busy, heavy internet we've gotten used to:
    • Mastodon - Decentralized social media A great Twitter/X alternative in these trying times. People here are friendly, and I've found I enjoy the experience of only seeing posts from people/tags I follow, as opposed to an algorithm.
    • Lemmy - Link aggregator for the Fediverse A Reddit alternative. Like Mastodon, I've also found the people here friendly and helpful, and there's plenty of niche and interesting communities to follow.
  • The Rust Programming Language One of the new memory-safe languages on the block these days, I've been enjoying learning it and like how it enforces very specific software design to minimize unexpected behavior through its super helpful compiler.
  • Elementary OS My first real experience with Linux and Open Source software. I still have a special place in my heart for this project and what it's done for improving user interface design in the Linux desktop space.
  • Fedora Linux I've settled on this Linux distribution as my main OS after my initial years of exploration, and it's been stable and comfortable. I wrote this website from my laptop running Fedora!
  • Pi-Hole Self-hosted DNS adblocker Nice little service to tinker with, handy for learning about DNS and networking. Has the added bonus of extra protection against more malware/phishing domains than ISPs usually provide with their DNS servers.
  • Syncthing - File syncing service Far and away one of the most useful services I run for myself. It's a handy solution for sharing folders and files that you want multiple devices to have, like a music collection or school notes.
  • Obsidian - Flexible writing app This is a Markdown notes editor/writing app/personal wiki, and I use it almost daily. It's well-suited for taking notes and connecting concepts with its cross-page linking, and the Canvas feature is great for building visual connections between notes and images while studing for an exam. Genuinely one of the most powerful apps I use!
